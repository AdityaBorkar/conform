# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly:

- **Do not** open a public GitHub issue
- Email the maintainer privately with details of the vulnerability
- Include steps to reproduce and the potential impact

We will acknowledge your report within 48 hours and aim to provide a fix or mitigation within 7 days.

## Disclosure Policy

- Vulnerabilities will be disclosed after a fix has been released
- We will credit reporters who wish to be acknowledged
